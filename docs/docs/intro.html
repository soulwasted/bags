BAGS — Design System

Based upon nomenclature by Brad Frost — Atomic Design.

Extended using design tokens are named entities that store visual design information. These are used in place of hard-coded values (such as hex values for color or pixels for spacing) in order to maintain a scalable, consistent system for UI development.
